Este script é para desinstalar alguns aplicativos indesejados dos celulares Samsung. Ele utiliza vários comandos do ADB para desinstalar os aplicativos. Alguns dos aplicativos que o script tenta desinstalar incluem o Samsung Push Service, TV Digital, Rádio FM, Game Optimizing Service, Game Launcher, Game Tools, GOS Legacy, Email, Microsoft Word, Excel, PowerPoint, Skype, Samsung Max e o Manual do Usuário. É recomendado que você leia o script e verifique se deseja desinstalar todos os aplicativos listados antes de executá-lo.

Pré-requisitos:

  • Um computador com um sistema operacional Windows compatível com os scripts BAT
  • Depuracão USB ativa no dispositivo alvo
  • Baixe a versão mais recente aqui
  • Outras versões (pode dar "conexão recusada")

Notas

O script pode demorar algum tempo para ser concluído, dependendo do dispositivo e da quantidade de aplicativos a serem desinstalados.

(Temporário) Se você estiver no Windows 8 ou mais recente, desative o Windows SmartScreen e o Windows Defender. Eles vão tentar apagar o instalador do script, por uma suspeita de malware. Mas não se preocupe, este script é 100% seguro.

Dicas e recomendações

  • Certifique-se de que o dispositivo esteja carregado o suficiente antes de iniciar o script;
  • Certifique-se de que você tem uma cópia de segurança dos dados do dispositivo antes de iniciar o script;
  • Certifique-se de ler e compreender o script antes de executá-lo, para garantir que você deseja desinstalar todos os aplicativos listados;
  • Certifique-se de que você tem todas as permissões necessárias para desinstalar os aplicativos listados;
  • Certifique-se de que você sabe como instalar manualmente no futuro qualquer aplicativo que você tenha desinstalado.
