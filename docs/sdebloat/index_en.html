Samsung Debloat

by Lucas Gabriel (lucmsilva)

This script is for uninstalling unwanted apps from Samsung device. It uses ADB commands to uninstall applications. It shows confirmation messages after each command to let you know if the uninstall was successful. Some of the applications that the script attempts to uninstall include Samsung Push Service, Digital TV, FM Radio, Game Optimizing Service, Game Launcher, Game Tools, GOS Legacy, Email, Microsoft Word, Excel, PowerPoint, Skype, Samsung Max, and User Manual. It is recommended that you read the script and make sure you want to uninstall all listed applications before running it.

Prerequisites:
  • A computer with a Windows operating system that supports BAT scripts
  • Active USB debugging on target device

Notes

The script may take some time to complete, depending on the device and the number of applications to be uninstalled.

(Temporary) If you are on Windows 8 or newer, disable Windows SmartScreen and Windows Defender. They will try to delete the script's installer, on suspicion of malware. But don't worry, this script is 100% safe.

Tips and recommendations
  • Make sure the device battery is fully charged before starting the script;
  • Make sure you have a backup of your device data before starting the script;
  • Be sure to read and understand the script before running it to ensure you want to uninstall all applications listed;
  • Make sure you have all necessary permissions to uninstall the listed apps;
  • Make sure you know how to manually install any apps you may have uninstalled in the future.
